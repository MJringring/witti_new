# 🌿 WITTI - 교사의 하루를 덜어주는 플랫폼

## 프로젝트 개요
- **이름**: WITTI (Wise Teachers' Interactive Tools & Insights)
- **목표**: 교사의 업무 부담을 줄이고 마음의 여유를 제공하는 통합 플랫폼
- **슬로건**: "출근길 5분, 위트 있는 인사이트 한 컷"

## 📍 사이트 구조

```
                    🌿 WITTI Platform
                           |
        ┌──────────────────┼──────────────────┐
        |                  |                  |
    [Home] ────────→  [Learn] ────────→  [Story]
      ↓                  ↓                  ↓
    [Talk] ←───────  [MyWITTI] ────────→ [Tools]
      ↓                  ↓                  ↓
   Community          Dashboard         AI Tools
```

**페이지 연결 흐름**:
- **Home** → Learn, Story, Talk 추천 콘텐츠로 연결
- **Learn** → 강의 수강 → MyWITTI에서 진행 관리
- **Story** → 공감/저장 → MyWITTI에 기록
- **Talk** → 프로젝트 참여 → MyWITTI에서 관리
- **Tools** → AI 도구 사용 → MyWITTI에 히스토리 저장
- **MyWITTI** → 모든 활동 통합 대시보드

## 주요 기능

### ✅ 현재 구현된 기능

#### 1. **홈페이지 (Home)** - 사용자 플로우 기반 리뉴얼
   - **오늘의 AI 추천 인사이트** (3개 카드)
   - **지금 인기 클래스** (평점 및 수강생 수 표시)
   - **교사 인터뷰 하이라이트** (실제 교사 사례)
   - **오늘의 질문** (공감 참여형 CTA) - "요즘 가장 힘든 순간은 언제인가요?"
   - 234명 실시간 참여 표시
   - 시작하기 버튼 (회원가입으로 연결)

#### 2. **Learn 페이지 - 실전형 교사 강의**
   - "5분만에 배우고, 내일 바로 써먹는 실전 콘텐츠"
   - 추천 클래스:
     - 🎓 AI로 부모면담 정리하기
     - 📋 놀이일지 쉽게 작성하기
     - 💬 감정케어 & 회복 클래스

#### 3. **Story 페이지 - 교사들의 이야기** ✨ 완전 리뉴얼
   - **상단 필터**: 🔹 전체 | 인터뷰 | 칼럼 | 영상 | 뉴스레터 (Sticky Navigation)
   - **3열 카드 레이아웃** (반응형: 3열 → 2열 → 1열)
   - **카드 구성**:
     - 썸네일 (그라데이션 배경 + 이모지 아이콘)
     - 제목 + 2줄 요약
     - AI 추천 배지 (오렌지)
     - 작성자 정보 + 좋아요/댓글 수
   - **카테고리별 콘텐츠**:
     - 🎤 인터뷰: "첫 출근, 그리고 첫 눈물", "20년 차 부장님의 조언"
     - ✍️ 칼럼: "번아웃에서 벗어나기", "디지털 시대의 교육법"
     - ▶️ 영상: "교실 속 작은 기적들", "선생님의 하루 VLOG"
     - 📧 뉴스레터: "이번 주 교육 트렌드", "마음을 덜어주는 월요일"
   - **뉴스레터 구독 섹션**:
     - "매주 월요일, 교사 마음을 덜어주는 뉴스레터"
     - 카카오 로그인 연동 버튼 (#FEE500 배경)
     - 구독자 수 표시: 12,847명
   - **인터랙션**:
     - 카테고리별 필터링 (JavaScript)
     - 호버 효과 (카드 상승 애니메이션)
     - 무한 스크롤 준비 (Lazy Loading)

#### 4. **Talk 페이지 - 교사 커뮤니티** ✨ 완전 리뉴얼
   - **탭 구조**: 🔹 공감 피드 | 실천 프로젝트 | 밋업 | 제안
   
   **공감 피드 탭**:
   - 소셜 피드 형태의 게시글 카드
   - 교사들의 일상 이야기와 경험 공유
   - 아바타 + 이름 + 직급 + 경력 표시
   - CTA: ❤️ 공감 / 💬 댓글 / 🔔 팔로우
   - 예시 게시글: "아이가 '선생님 좋아해요'라고...", "부모상담 성공 후기"
   
   **실천 프로젝트 탭**:
   - 소규모 그룹 (4~6인) 활동 리스트
   - 왼쪽 오렌지 테두리 강조
   - 모집 상태 배지 (모집 중/진행 중)
   - 참여자 수, 기간, 모임 빈도 표시
   - 프로젝트 예시:
     - "부모상담 개선 실험" (4/6명 모집 중)
     - "교실 놀이연구회" (6/6명 진행 중)
     - "AI 도구 마스터 챌린지" (2/6명 모집 중)
   
   **밋업 탭**:
   - 오프라인 행사 일정 카드
   - 날짜 블록 (월/일) + 행사 정보
   - 위치 (지도 아이콘) + 참석자 수
   - 참가 신청 / 알림 받기 버튼
   - 예시: "서울 교사 네트워킹 모임", "부산 신규교사 멘토링 데이"
   
   **제안 탭**:
   - 주제 제안 게시판
   - 투표 시스템 (👍 추천 버튼 + 카운트)
   - 제안 내용 + 제안자 정보
   - 예시: "학급 경영 노하우 시리즈", "학부모 소통 메시지 템플릿"
   
   **공통 기능**:
   - Sticky 탭 네비게이션 (스크롤 시 상단 고정)
   - 탭 전환 JavaScript 인터랙션
   - 우하단 플로팅 글쓰기 버튼 (✏️)
   - 반응형 디자인 (모바일 최적화)

#### 5. **Tools 페이지 - AI 도구 허브** ✨ 완전 리뉴얼
   - **2×2 도구 대시보드** 그리드 레이아웃
   - **인터랙티브 AI 워크스페이스**
   - **My Toolkit 저장소** (히스토리 + 템플릿)
   
   **4개 핵심 AI 도구**:
   
   1. **🤖 부모면담 요약기** (가장 인기)
      - 긴 면담 내용을 AI가 핵심만 추려서 정리
      - 면담 기록 시간 90% 단축
      - 출력 예시: 일시/대상/주요내용/조치사항/후속계획
   
   2. **📝 일일일지 자동작성** (시간 절약)
      - 오늘 활동을 간단히 입력하면 체계적인 일지로 변환
      - 출력 예시: 날짜/날씨/활동내용/특이사항/교사소견
   
   3. **💗 감정일지 & 마음진단** (마음 건강)
      - 교사의 감정 상태 체크 및 번아웃 예방
      - WITTI Care 시스템
      - 출력 예시: 감정상태/번아웃위험도/추천케어/장기관리
   
   4. **📊 성장 리포트 생성기** (PDF 출력)
      - 아이의 발달 과정을 PDF 리포트로 자동 생성
      - 학부모 공유용
      - 출력 예시: 발달영역별 평가/종합의견
   
   **AI 워크스페이스**:
   - 도구 선택 시 동적 헤더 변경 (아이콘 + 제목)
   - 대형 텍스트 입력창 (200px, 플레이스홀더 예시)
   - AI 정리 버튼 (그라데이션 오렌지) + 초기화 버튼
   - 결과 섹션 (요약 텍스트 + 복사 버튼)
   - AI 처리 시뮬레이션 (1.5초 로딩)
   - 자동 스크롤 및 active 상태 관리
   
   **My Toolkit 섹션**:
   - **최근 사용 도구** (3개):
     - 아이콘 + 도구명 + 사용 시간
     - 호버 시 배경색 변경 + 우측 슬라이드
   - **내 템플릿 저장소** (3개):
     - 학부모 상담 템플릿
     - 월간 활동 보고서
     - 발달 체크리스트
     - 3열 그리드 (반응형)

#### 6. **MyWITTI 페이지 - 나의 성장 공간** ✨ 완전 리뉴얼
   - **개인 대시보드** 형태의 통합 관리 페이지
   
   **👤 프로필 섹션**:
   - 120px 원형 프로필 사진
   - 이름 + 직무 + 경력 표시 (예: "유치원 교사 · 3년차")
   - 성장 레벨 배지: "⭐ Lv.3 주임교사"
   - 그라데이션 배경 (#ffe9d6 → #fff0e6)
   
   **📊 나의 활동 요약** (3개 통계 카드):
   - 수강한 강의: **12개** (큰 숫자 강조)
   - 공감한 이야기: **37개**
   - 획득한 뱃지: **5개**
   - 호버 시 상승 효과 + 오렌지 테두리
   - 클릭 시 상세 페이지 이동
   
   **🌳 성장 트리 시각화**:
   - 5개 스킬 브랜치 (가로 배치, 300px 높이):
     1. 🤝 부모상담 (노드 2개 해제)
     2. 🎨 놀이지도 (노드 3개 해제)
     3. 🤖 AI 활용 (노드 1개 해제)
     4. 💗 감정케어 (노드 2개 해제)
     5. 👑 리더십 (노드 1개 해제)
   - 각 브랜치당 3개 노드 (60px 원형)
   - 해제된 노드: 그라데이션 배경 + 이모지 + 오렌지 테두리
   - 잠긴 노드: 회색 + 🔒 아이콘
   - 클릭 시 관련 콘텐츠 연결
   
   **🎓 나의 클래스 관리** (4개 카드):
   - **수강 중**: "AI로 부모면담 정리하기"

     - 상태 배지: 진행중

     - 진행률 바: 100% (15/15 강의)
     - 상태 배지: 완료 (녹색)
   - **참여 프로젝트**: "부모상담 개선 실험"
     - 정보: 4명과 함께 · 4주 프로젝트 2주차
     - 상태 배지: 참여중
   - **수강 예정**: "감정케어 & 회복 클래스"
     - 시작일: 2025년 2월 1일
     - 상태 배지: 예정 (회색)
   - 2열 그리드 (반응형: 1열)
   
   **🤝 AI 추천 멘토** (3명):

     - 초등학교 교사 · 7년차

   - 80px 아바타 + 멘토 신청 버튼
