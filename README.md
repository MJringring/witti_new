# 🌿 WITTI - 교사의 하루를 덜어주는 플랫폼

## 프로젝트 개요
- **이름**: WITTI (Wise Teachers' Interactive Tools & Insights)
- **목표**: 교사의 업무 부담을 줄이고 마음의 여유를 제공하는 통합 플랫폼
- **슬로건**: "출근길 5분, 위트 있는 인사이트 한 컷"

## 주요 기능

### ✅ 현재 구현된 기능

#### 1. **홈페이지 (Home)** - 사용자 플로우 기반 리뉴얼
   - **오늘의 AI 추천 인사이트** (3개 카드)
   - **지금 인기 클래스** (평점 및 수강생 수 표시)
   - **교사 인터뷰 하이라이트** (실제 교사 사례)
   - **오늘의 질문** (공감 참여형 CTA) - "요즘 가장 힘든 순간은 언제인가요?"
   - 234명 실시간 참여 표시
   - 시작하기 버튼 (회원가입으로 연결)

#### 2. **Learn 페이지 - 실전형 교사 강의**
   - "5분만에 배우고, 내일 바로 써먹는 실전 콘텐츠"
   - 추천 클래스:
     - 🎓 AI로 부모면담 정리하기
     - 📋 놀이일지 쉽게 작성하기
     - 💬 감정케어 & 회복 클래스

#### 3. **Story 페이지 - 교사들의 이야기** ✨ 완전 리뉴얼
   - **상단 필터**: 🔹 전체 | 인터뷰 | 칼럼 | 영상 | 뉴스레터 (Sticky Navigation)
   - **3열 카드 레이아웃** (반응형: 3열 → 2열 → 1열)
   - **카드 구성**:
     - 썸네일 (그라데이션 배경 + 이모지 아이콘)
     - 제목 + 2줄 요약
     - AI 추천 배지 (오렌지)
     - 작성자 정보 + 좋아요/댓글 수
   - **카테고리별 콘텐츠**:
     - 🎤 인터뷰: "첫 출근, 그리고 첫 눈물", "20년 차 부장님의 조언"
     - ✍️ 칼럼: "번아웃에서 벗어나기", "디지털 시대의 교육법"
     - ▶️ 영상: "교실 속 작은 기적들", "선생님의 하루 VLOG"
     - 📧 뉴스레터: "이번 주 교육 트렌드", "마음을 덜어주는 월요일"
   - **뉴스레터 구독 섹션**:
     - "매주 월요일, 교사 마음을 덜어주는 뉴스레터"
     - 카카오 로그인 연동 버튼 (#FEE500 배경)
     - 구독자 수 표시: 12,847명
   - **인터랙션**:
     - 카테고리별 필터링 (JavaScript)
     - 호버 효과 (카드 상승 애니메이션)
     - 무한 스크롤 준비 (Lazy Loading)

#### 4. **Talk 페이지 - 교사 커뮤니티** ✨ 완전 리뉴얼
   - **탭 구조**: 🔹 공감 피드 | 실천 프로젝트 | 밋업 | 제안
   
   **공감 피드 탭**:
   - 소셜 피드 형태의 게시글 카드
   - 교사들의 일상 이야기와 경험 공유
   - 아바타 + 이름 + 직급 + 경력 표시
   - CTA: ❤️ 공감 / 💬 댓글 / 🔔 팔로우
   - 예시 게시글: "아이가 '선생님 좋아해요'라고...", "부모상담 성공 후기"
   
   **실천 프로젝트 탭**:
   - 소규모 그룹 (4~6인) 활동 리스트
   - 왼쪽 오렌지 테두리 강조
   - 모집 상태 배지 (모집 중/진행 중)
   - 참여자 수, 기간, 모임 빈도 표시
   - 프로젝트 예시:
     - "부모상담 개선 실험" (4/6명 모집 중)
     - "교실 놀이연구회" (6/6명 진행 중)
     - "AI 도구 마스터 챌린지" (2/6명 모집 중)
   
   **밋업 탭**:
   - 오프라인 행사 일정 카드
   - 날짜 블록 (월/일) + 행사 정보
   - 위치 (지도 아이콘) + 참석자 수
   - 참가 신청 / 알림 받기 버튼
   - 예시: "서울 교사 네트워킹 모임", "부산 신규교사 멘토링 데이"
   
   **제안 탭**:
   - 주제 제안 게시판
   - 투표 시스템 (👍 추천 버튼 + 카운트)
   - 제안 내용 + 제안자 정보
   - 예시: "학급 경영 노하우 시리즈", "학부모 소통 메시지 템플릿"
   
   **공통 기능**:
   - Sticky 탭 네비게이션 (스크롤 시 상단 고정)
   - 탭 전환 JavaScript 인터랙션
   - 우하단 플로팅 글쓰기 버튼 (✏️)
   - 반응형 디자인 (모바일 최적화)

#### 5. **Tools 페이지 - AI 도구 허브**
   - 교사를 위한 AI 도구 허브
   - 일상 업무를 AI로 간편하게
   - 주요 도구:
     - 🤖 부모면담 요약기
     - 📝 일일일지 자동작성
     - 💗 감정일지 & 마음진단 (WITTI Care)
     - 📊 성장 리포트 생성기
     - 📁 나만의 템플릿 저장소

#### 6. **MyWITTI 페이지 - 나의 성장 공간**
   - 배움의 기록부터 성취까지
   - 나의 활동:
     - 📚 수강 중인 강의 관리
     - ⭐ 저장한 콘텐츠
     - 🌱 성장 트리 & 뱃지
   - 나의 참여:
     - 🎯 실천 프로젝트 관리
     - 👥 멘토·멘티 매칭
     - 🎓 나의 클래스 (개설자 전용)
   - 프로필 관리:
     - 👤 역량 및 관심 분야
     - 📈 성장 리포트
     - 🔔 맞춤형 알림 설정

#### 7. **회원가입/로그인 시스템**
   - 이메일 기반 회원가입
   - 직무 선택 (신입/주임/선임/원장/학생)
   - 관심 키워드 선택 (놀이/상담/AI/성장/리더십/발달)
   - 카카오 로그인 연동 준비
   - 폼 유효성 검사

#### 8. **온보딩 페이지**
   - AI 맞춤 콘텐츠 3개 추천
   - 직무별 맞춤형 제안
   - 나만의 WITTI 루틴 시작
   - 홈으로 부드러운 전환

#### 9. **RESTful API**
   - `/api/hello` - API 상태 확인
   - `/api/insights` - 인사이트 데이터 JSON 제공

### 설치
```bash
cd /home/user/webapp
npm install
```

### 개발 서버 실행
```bash
# 빌드
npm run build

# PM2로 서버 시작
pm2 start ecosystem.config.cjs

npm run dev:sandbox
```

```bash
# 홈페이지 테스트
curl http://localhost:3000

# API 테스트

curl http://localhost:3000/api/insights
```

```bash
# 상태 확인
pm2 list

# 로그 확인
pm2 logs witti --nostream

# 재시작
pm2 restart witti

# 중지
pm2 stop witti

# 삭제
pm2 delete witti
```
